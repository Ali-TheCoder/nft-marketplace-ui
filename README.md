# NebulaMart — NFT Marketplace UI

A futuristic NFT marketplace front-end built with **Next.js (App Router)**, **TypeScript**, and **Tailwind CSS**.

## Features
- Premium sci-fi UI theme
- Market page with search / filter / sort (client-side)
- NFT detail page with tabs
- Profile page with stats and collections
- Smooth animations (Framer Motion)

## Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Run locally
```bash
npm install
npm run dev
